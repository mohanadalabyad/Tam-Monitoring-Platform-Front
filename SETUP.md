# TAM Monitoring Platform - Installation & Setup Guide

## Quick Start

Follow these steps to get the TAM Monitoring Platform up and running:

### 1. Install Dependencies

```powershell
npm install
```

This will install all the required Angular and development dependencies.

### 2. Start the Development Server

```powershell
npm start
```

The application will open automatically in your default browser at `http://localhost:4200`.

### 3. Build for Production

```powershell
npm run build
```

The production-ready files will be in the `dist/tam-monitoring-platform` directory.
